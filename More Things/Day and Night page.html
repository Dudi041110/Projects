What's the game?

It's a 2D platformer that stars an unnamed cube. The story is still a WIP but the core game is done. Basically you've got to venture through worlds fighting bosses and enemies for some undecided reason. The way you fight currently is by punching enemies using JKL and it's planned for more fighting mechanics to be added, but first I've got to perfect the default.
